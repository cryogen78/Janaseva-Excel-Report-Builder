Janaseva Excel Report Builder
Purpose
Enterprise reporting platform for banks, cooperative societies, finance departments, auditors, branch managers, and administrative users. It enables Excel data uploads, report generation, analytics, user management, templates, and export capabilities from a centralized web application.
Target Users
•	Bank Administrators
•	Branch Managers
•	Finance & Accounts Teams
•	Internal Auditors
•	Management & Decision Makers
•	Data Entry and Operations Staff
Core Features
•	Secure login and role-based access control
•	Dashboard with reporting statistics and recent activity
•	Excel (.xlsx, .xls) and CSV file upload
•	Multi-sheet Excel processing
•	Excel data preview before reporting
•	Dynamic report builder
•	Column selection
•	Filtering and data slicing
•	Group By operations
•	SUM, AVG, COUNT, MIN and MAX calculations
•	Template management
•	Saved report management
•	Analytics dashboards and charts
•	User management
•	Report history tracking
•	Audit logging
•	Scheduled reporting support
•	Excel export
•	PDF export readiness
•	Branch-wise reporting
•	Loan portfolio analysis
•	Deposit analysis
•	Financial ratio calculations
Workflow
•	Login using authorized credentials.
•	Upload Excel or CSV files.
•	Preview uploaded sheets and columns.
•	Select report columns.
•	Apply filters and business rules.
•	Configure calculations and aggregations.
•	Group data by branch, department, product, or other fields.
•	Preview generated results.
•	Save report definitions as templates.
•	Export reports to Excel or PDF.
•	Review historical reports and analytics.

Business Benefits
•	Reduces manual reporting effort
•	Improves reporting accuracy

•	Provides faster management insights
•	Supports audit and compliance requirements
•	Centralizes reporting operations
Technical Overview
Frontend: React.js
Backend: Node.js / Express
Database: SQLite
File Processing: Excel and CSV ingestion
Visualization: Interactive analytics dashboards and charts
Recommended Usage
Use this platform as the central reporting engine for operational, financial, loan, deposit, audit, compliance, and management reporting requirements.
